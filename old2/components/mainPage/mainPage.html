Интерактивный учебник

Здесь вы можете изучить материал и проверить свои занния.

Данный сервис создан для быстрого создания и оценки зананий по любым темам.

Приложение состоит из двух частей:

  • Учебно-методическое пособие
  • Раздел оценки полученных знаний

Вы можете загрузить материалы в формате Microsoft Word, а на выходе получить интерактивный учебник и средства контроля полученных знаний, для этого необходимо проверить содержание документа определенному формату

Вы можете использовать сервис для самоконтроля или для контролья качества изученного материала студентами
